### 编译环境

|操作系统   |ubuntu 18.04	|
|---|---|
|开发框架   |Qt 5.12.5		|
|语言		|C++,QML		|
|运行环境   |aarch64-linux	|
|交叉编译链 |gcc-linaro-7.3.1-2018.05-x86_64_aarch64-linux-gnu |

+ 由于使用了SemiDrive的libSdCast.so进行仪表投屏功能，目前仅支持运行在aarch64环境，如果不需要此功能，也可运行在windows/x86_64_linux。
+ 需要交叉编译Qt 5.12.5源码
